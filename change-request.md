# Tool: Project Change Request

**Project Name:** Personal Introduction Website  
**Request By:** Cheri Whalen, M.S. (my CMPA 3301 Instructor)

## Proposed Change
Develop and add an embedded Google Map feature to the website that displays the location of my hometown. The map would identify Farwell, TX as my original hometown. The feature would be placed on the About page and require selecting the proper location view, embedding the map, and formatting it so it fits the current site design. There potentially would need to be editing of the color scheme as well.

## Reasons for the Change
The stakeholder wants the website to include a map based feature that makes the site more interactive and gives visitors visual representation of important location connected to my background. Instead of only reading text about where I went to school, where I work, and where I live, etc. the visitors would be able to see my hometown location directly on the site.

## How This Change Will Add Value
This change would add value by making the website more engaging, more personal, and easier to connect with visually. A map showing Farwell, TX would help tell the story of where I started and help readres to see how my education, career, and current life began. It would also make the site more polished because it's currently text only and needs to be "spiced up".

## How This Change Will Affect the Constraints

**Time**  
I estimate that this change would have a low to moderate impact on time. The feature itself is not extremely difficult, but it would definitely take extra time to embed the map correctly, test that it works, and make sure it fits the design of the rest of the website.

**Quality**  
If implemented well, this change could improve the overall quality of the website by adding a visual and interactive element. It would make the site feel less plain. However, if it is not formatted well, it could make the page feel rushed and could potentially distract from the main content.

**Budget**  
I estimate that this change would have little to no budget impact. For this student project, the main cost would be time and effort rather than money.  I am confident a simple embedded Google Map can be added without extra expense.
